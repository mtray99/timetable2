# 🎓 AI 시간표 생성기

AI 기반 최적 시간표 생성 시스템. 학년/학기, 과목을 선택하고 조건을 설정하면 자동으로 최적의 시간표를 생성합니다.

## ✨ 기능

- **학년/학기별 커리큘럼 자동 로드**
- **다중 선택지 지원** - 각 과목별 여러 수업 시간 중 선택
- **스마트 스케줄링** - 시간표 충돌 자동 감지 및 회피
- **조건 설정**
  - 🗓️ 공강 요일 지정
  - 🌅 아침 수업 제외
  - 🍽️ 점심 시간 확보
  - 💛 선호/기피 교수 지정
- **최적화 스코링** - 학점, 공강일, 수업 시간 균등 분배 등 고려
- **다중 결과** - 최대 5개의 최적 조합 제시

## 🚀 빠른 시작

### 1. 설치

```bash
# 필수 패키지 설치
pip install -r requirements.txt
```

### 2. 서버 실행

```bash
# 개발 모드 (자동 재로드)
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# 또는 프로덕션 모드
python main.py
```

### 3. 웹 접속

- 로컬: `http://localhost:8000`
- 네트워크: `http://<서버IP>:8000`

## 📊 시간표 생성 알고리즘

### 제약 조건 검증
- 시간 충돌 감지
- 공강일 확보
- 학점 범위 (0-21학점)
- 교수 선호도

### 스코링 시스템
1. **공강일 보너스** (20점) - 수업이 없는 요일 많을수록 높음
2. **빈 시간 최소화** (25점) - 같은 날 수업 사이 시간 차이 최소화
3. **목표 학점 근접** (20점) - 목표 학점(18학점)에 가까운 정도
4. **아침 수업 회피** (15점) - 아침 수업 최소화
5. **선호 교수 보너스** (5점) - 선호 교수 강의 추가 점수
6. **수업 시간 균등** (10점) - 요일별 수업 시간 분산

**총점:** 최대 100점

## 🛠️ 커스터마이징

### 데이터 수정

`internal_data.json`에서:
- `curriculum`: 학년/학기별 과목 추가/삭제
- `lectures`: 강의 정보 (교수, 시간, 강의실 등)

```json
{
  "curriculum": {
    "1-1": [
      {
        "name": "과목명",
        "category": "분류",
        "required": false,
        "lecture_ids": [0, 1, 2]
      }
    ],
    "lectures": [
      {
        "id": 0,
        "name": "강의명",
        "professor": "교수명",
        "credits": 3.0,
        "time_slots": [...]
      }
    ]
  }
}
```

### 스코링 가중치 조정

`solver.py`의 `score_schedule()` 함수에서 점수 계수 수정

## 🐛 문제 해결

### 포트 점유 오류

```bash
python -m uvicorn main:app --host 0.0.0.0 --port 8001
```

### 시간표 생성 실패

1. 최소 과목 1개 선택 확인
2. 조건 완화 (공강일 해제, 아침 제외 해제)
3. 백엔드 로그에서 에러 메시지 확인

### CORS 에러

- 이미 모든 출처 허용으로 설정됨
- 브라우저 개발자 도구(F12) > Network에서 API 응답 확인
